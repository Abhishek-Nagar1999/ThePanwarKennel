# The Panwar Kennel — Premium Dog Kennel Website

A modern, mobile-first React website for a dog kennel and puppy business in Delhi, India.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
npm run preview
```

## Customize Business Details

Edit `src/config.js` to update:

- Phone number & WhatsApp
- Instagram handle & URL
- Email & address
- Google Maps embed & link
- Service areas

## Features

- Premium minimal UI (beige, gold, clean typography)
- Sticky navbar with WhatsApp CTA
- Puppy cards with breed enquiry buttons
- Why Choose Us, About, Reviews, Contact, FAQ
- Floating WhatsApp button
- SEO meta tags + LocalBusiness & FAQ schema
- Mobile-responsive layout

## Tech Stack

- React 19 + Vite
- Vanilla CSS (no UI framework — fast & lightweight)
- Semantic HTML for SEO
